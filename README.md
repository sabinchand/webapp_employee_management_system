# webapp_employee_management_system
WEB APP - EMPLOYEE MANAGEMENT SYSTEM

FEATURES

    1. Save a employee (C)
    2. Read the employees (R)
    3. Update the employee (U)
    4. Delete the employee (D)
    
 DEPDENCIES USED
 
    1. Spring Web
    2. MySQL Driver
    3. Spring Data JPA
    4. Thymeleaf
    5. Spring Boot DevTools
